Expression Evaluation using Stack 🚀

    A futuristic DSA Visualizer for understanding Expression Evaluation using Stack, including:
    
    Infix ➜ Postfix Conversion
    Infix ➜ Prefix Conversion
    Postfix Evaluation
    Stack Operations Visualization
    Step-by-Step Animation
    Pseudocode Highlighting
    Interactive Learning UI
    
    Built using HTML, CSS, and JavaScript with neon cyberpunk styling and animations.

What is a Stack?
A linear data structure following LIFO (Last In, First Out). Think of plates stacked on top of each other.

Shunting Yard Algorithm
Invented by Edsger Dijkstra, it parses infix equations into postfix (Reverse Polish) logic.

Time & Space Complexity
Time: O(n) since each token is read exactly once, and Operators pushed/popped at most once.
Space: O(n) to store output and stacks.
